# Serverless Tickets
Simple repo of the steps/procedures I took to create a Serverless Ticketing System using AWS Lambda, DynamoDB, and API Gateway. 

📌 Due to test prep burnout and a feeling of not labbing enough I began working on a Serverless Self Healing Ticketing System. This project will help by giving me hands-on 
experience using a variety of AWS services:
 - AWS Lambda - Severless Functions
 - API Gateway - 
 - Cloudwatch - Monitoring
 - IAM - for Access management (Roles)
 - Secrects Manager - To prevent hardcoding any pertinent information

🚀 I began by first logging into the AWS console and creating a new DynamoDB Table, giving it a name and partitiion key but keeping all other settings set to default
